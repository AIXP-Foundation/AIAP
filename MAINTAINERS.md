# Maintainers

The AIAP Protocol is maintained by the AIAP Protocol Organization across three governance domains.

## Core Maintainers

| Name | Domain | Scope | Contact |
|------|--------|-------|---------|
| AIAP Protocol Organization | aiap.dev | Protocol specification, quality standards | [GitHub](https://github.com/AISOP-AIAP-SoulBot/AIAP) |

## Domain Responsibilities

| Domain | Files | Role |
|--------|-------|------|
| `aisop.dev` | `.aisop.json` format definition | Format steward |
| `aiap.dev` | `specification/`, `docs/`, `adrs/` | Governance steward |
| `soulbot.dev` | Reference runtime implementation | Executor steward |

## Becoming a Maintainer

Contributors who have demonstrated sustained, high-quality contributions to the AIAP protocol may be nominated as maintainers by existing maintainers. See [CONTRIBUTING.md](CONTRIBUTING.md) for details.
